About Landing Page

Landing Page is a simple HTML/CSS/JS single page website built on Bootstrap. You can use this static site as a starting point for a simple website or something bigger.

Just 3 Files?

Landing Page consists of just 3 files: index.html (this page), custom.css, and main.js. There are 2 folders. A css folder for Cascading Style Sheets and a js folder for Javascript. There is not very much code in these files and they are ready to be edited.

index.html

This is the HTML part of the app that actually displays content such as this text and the other elements on this page. This is also where the website's dependencies are declared such as Bootstrap, Tether, and jQuery.

custom.css

This is the Cascading Style Sheets file that applies styles to the HTML in index.html. This is run right at the end of index.html.

main.js

This is where the Javascript used to transition between parts of the page is stored.

Dependencies

As previously mentioned Landing Page depends on Bootstrap, jQuery, and Tether which are referenced as CSS and Javascript files in index.html. Landing Page also depends on Font Awesome which comes with Bootstrap and is referenced in the head element of index.html. Landing Page uses 5 CDN links in total: Font Awesome and Bootstrap CSS, Bootstrap, jQuery, and Tether Javascript. These could easily be replaced with local versions.

Sunflowers (original title, in French: Tournesols) are the subject of two series of still life paintings by the Dutch painter Vincent van Gogh. The earlier series, executed in Paris in 1887, depicts the flowers lying on the ground, while the second set, executed a year later in Arles, shows bouquets of sunflowers in a vase. In the artist's mind both sets were linked by the name of his friend Paul Gauguin, who acquired two of the Paris versions.
